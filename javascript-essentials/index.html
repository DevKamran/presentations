Objects in JavaScript

  • An object is an unordered collection of key-value pairs.

var person = {
   first: 'Travis',

};

Functions in JavaScript

  • Functions are first-class objects.
  • Functions can be anonymous.
  • Functions can be "self-executing"
  • Functions encapsulate and capture context.

Function Closures


var say = function(what) {
   return function() {
      console.log(what);
   };
};

setTimeout(say('hello'), 1000);
setTimeout(say('there'), 2000);

Modules


var Module = (function() {
   var privateVar = 'secret';
   var privateMethod = function() {
      console.log('I am private!');
   };

   return {
      publicVar: 'public',
      publicMethod: function() {
         console.log('I am public!');
      }
   };
})();

Module.publicMethod();
console.log(Module.publicVar);

Common.JS Modules

Used in Node.js and other JS libraries.

splitter.js
module.exports = function(options) {
   options = options || {};
   options.separator = options.separator || '-';
   return {
      split: function(input) {
         return input.split(options.separator);
      }
   };
};
					
index.js

   separator: '_'
});

Understanding 'this'

  • http://bit.ly/VHXAYB
  • http://bit.ly/1SourKC
  • Difference between context and scope.
  • 'new' objects create context
  • 'this' is assigned to the object instance that 'invokes' a method.
  • You can bind context with 'bind'
  • You can call a method with context using 'call' or 'apply'

Object Oriented JavaScript

Prototype Inheritance

  • The prototype serves as the 'blueprint' for object instances.
  • Each object instance "inherits" the prototype.
  • Class extending supported through prototype cloning.

JavaScript Module Class


var Person = (function() {

   var privateVar = 'private';
   var privateMethod = function() {
      console.log('I am private!');
   };

   var Person = function(first, last) {
      this.first = first;
      this.last = last;
   };

   Person.prototype.getName = function() {
      return this.first + ' ' + this.last;
   };

   return Person;
})();

var person = new Person('Travis', 'Tidwell');
console.log(person.getName());
